## Recording checklist

- [ ] Intro — name, student #, title (≤ 30 s)
- [ ] Workflow overview and business rules
- [ ] Version A live demo
  - [ ] Submit under $100 → auto-approved
  - [ ] Submit ≥ $100 → manager approves
  - [ ] Submit ≥ $100 → manager rejects
  - [ ] Submit ≥ $100 → no response → escalated
  - [ ] Missing field → validation error
  - [ ] Invalid category → validation error
- [ ] Version B live demo
  - [ ] Publish message to Service Bus queue
  - [ ] Show Logic App run history (approved / rejected / escalated branches)
  - [ ] Show topic subscriptions receiving filtered messages
  - [ ] Show employee notification email from the notifier Logic App
- [ ] Walk through the comparison scoreboard (slide 8)
- [ ] Cost estimate (slide 10)
- [ ] Recommendation (slide 12)
- [ ] Lessons learned (slide 13)
- [ ] Close — repo URL visible on screen (≤ 30 s)

## Recording settings

- 1920 × 1080 (or higher), 30 fps minimum.
- Webcam overlay optional but recommended for the intro / close.
- Use the attached `slides.pptx` in presenter mode so speaker notes are visible.
